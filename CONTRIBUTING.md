# Contributing

Thanks for your interest in improving the MTConnect Ignition module.

## Quick Start
- Use JDK 17.
- Build the module: `./gradlew clean :mtconnect-build:build`
- Run tests: `./gradlew test` (no tests currently)

## Development Notes
- Keep changes focused and avoid large refactors in the same PR unless required.
- If you change behavior or configuration, update `README.md`.
- Validate changes in an Ignition 8.3 gateway when possible.

## Submitting Changes
1. Open an issue to discuss significant changes before starting.
2. Create a branch and open a PR with a clear description of the change.
3. Ensure the build passes before requesting review.
